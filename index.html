Welcome to Visualizing Shortest Path v.1.0!



This project uses the P5.js library to visualize pathfinding algorithms. Right now the only algorithm is Breadth-First Search, but A* will be implemented very soon!

To use this tool, simply "Place Start", "Place End", click on the grid to make obstacles, then click "Run Algorithm".
